# Fistflare Particles

一个基于 Three.js 和 MediaPipe Hands 的实时交互 3D 粒子系统。

## 功能

- 摄像头检测单手手势。
- 握拳：粒子发散飞向屏幕四周。
- 张开手掌、未检测到手或其他状态：粒子聚拢形成模型。
- 支持切换爱心、圆球、星环模型。
- 支持全屏显示。

## 环境要求

- Node.js 18 或更高版本。
- 支持 WebGL 的现代浏览器，推荐 Chrome 或 Edge。
- 一台可用摄像头。
- 首次运行需要联网安装依赖，并加载 MediaPipe 手势模型。

## 在另一台电脑运行

先安装 Node.js 18 或更高版本：

https://nodejs.org/

然后在项目目录打开终端，运行：

```bash
npm install
npm run dev
```

终端会显示本地地址，通常是：

```text
http://127.0.0.1:5173/
```

用浏览器打开这个地址，允许摄像头权限后即可体验。

## Windows 快速启动

在 Windows 上也可以双击：

```text
run-local.bat
```

如果是第一次运行，脚本会自动执行 `npm install`，然后启动本地服务。

## 打包构建

需要生成静态构建产物时运行：

```bash
npm run build
```

构建结果会输出到：

```text
dist/
```

如果只是在本机体验，使用 `npm run dev` 更方便。

## 使用方法

1. 打开本地服务地址。
2. 浏览器询问摄像头权限时选择允许。
3. 把一只手放到摄像头画面中。
4. 张开手掌或移开手：粒子聚拢成当前模型。
5. 握拳：粒子向屏幕四周发散。
6. 使用左侧面板切换爱心、圆球、星环模型。
7. 点击右上角全屏按钮进入或退出全屏。

## 注意

- 摄像头功能需要浏览器授权。
- 本地 `localhost` / `127.0.0.1` 可以使用摄像头权限。
- 如果部署到公网，必须使用 HTTPS，摄像头功能才会正常启用。
- 如果手势模型一直显示加载中，请检查网络是否能访问 MediaPipe 模型资源。
