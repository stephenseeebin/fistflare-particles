# Fistflare Particles

基于 Three.js 和 MediaPipe Hands 的实时 3D 粒子手势交互场景。

## 功能

- 调用电脑摄像头识别单手手势。
- 握拳：粒子快速向屏幕四周发散破碎。
- 张开手掌、未检测到手或其他状态：粒子聚拢成当前模板。
- 支持模板：爱心、花朵、土星、烟花、宇宙星河。
- 支持主光/辅光颜色拾取器，实时改变霓虹发光配色。
- 支持发散力度、发光强度调节。
- 支持手动聚拢/发散按钮，方便无摄像头时演示。
- 支持全屏显示。

## Windows 快速启动

先安装 Node.js 18 或更高版本：

https://nodejs.org/

解压项目后，双击：

```text
run-local.bat
```

脚本会在首次运行时自动安装依赖，然后启动本地服务并打开：

```text
http://127.0.0.1:5173/
```

打开后允许摄像头权限即可体验。

## 命令行运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建结果输出到 `dist/`。

## 注意

- 摄像头功能需要浏览器授权。
- 本地 `localhost` / `127.0.0.1` 可以使用摄像头权限。
- 首次运行需要联网安装依赖，并加载 MediaPipe 手势模型。
- 如果部署到公网，必须使用 HTTPS，摄像头功能才会正常启用。
